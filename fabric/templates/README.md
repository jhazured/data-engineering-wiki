# Templates

This folder contains templates and checklists for implementing the T0-T5 architecture pattern.

## Contents

### Project Setup Checklist
- **File**: `project-setup-checklist.md`
- **Description**: Step-by-step checklist for setting up a new T0-T5 project
- **Use Case**: Starting a new project

### Architecture Decision Record Template
- **File**: `adr-template.md`
- **Description**: Template for documenting architecture decisions
- **Use Case**: Documenting key architectural choices

### Incident Response Playbook
- **File**: `incident-response-playbook.md`
- **Description**: Playbook for responding to incidents in production
- **Use Case**: Operations and troubleshooting

## Usage

1. Copy the template file
2. Customize for your project
3. Store in your project repository
4. Update as needed
